휴고 블로그 만들기

이 글은 Hugo라는 정적 사이트 생성기(Static site generator)를 이용해 깃헙 페이지로 블로그를 개설하는 방법을 소개한다. Hugo 설치 휴고 공식 사이트에서 설치 방법을 참고하여 사용 중인 OS에 맞게 설치한다. 바이너리 파일을 받거나, 패키지 매니저로 설치할 수 있다. 여기서는 Windows에서 Chocolatey 패키지 매니저를 통해 설치한다. choco install hugo -confirm 설치가 완료되었다면 hugo 커맨드를 사용할 수 있다. hugo version 커맨드를 입력하여 버전을 확인해볼 수 있다. hugo version hugo v0.102.1-e9c3ce16abc391ee5be38f88b0499d0e8e19104c windows/amd64 BuildDate=2022-08-29T12:58:13Z VendorInfo=gohugoio 블로그 만들기 초기화 적당한 디렉토리에 hugo new site blog 커맨드를 입력한다.
